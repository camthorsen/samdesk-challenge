# Samdesk code challenge

Solution to the [Advent of Code](https://adventofcode.com/2024/day/2) puzzle (Day 2), written in TypeScript.

## Prerequisites

- [Node.js](https://nodejs.org) 20+
- [pnpm](https://pnpm.io)

## Setup

```bash
pnpm install
```

## Running a solution

```bash
npx ts-node getSafeReportCount.ts
```

The solution reads from `input.txt` in the project root.

## Running tests

```bash
pnpm test
```
